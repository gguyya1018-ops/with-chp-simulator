# 운영손익 계산

> 운전최적화 시스템의 계산 로직 기준으로 정리
> 기동계획(위드시뮬) → 운영손익 산출로 연결
> 마지막 갱신: 2026-03-15

---

## 1. 전체 손익 구조

```
운영손익 (공헌이익) = 매출 - 재료비

[매출]
├── 전기판매
│   ├── 급전매출        = Σ(QPC×MW² + LPC×MW + NLPC)  (시간별)
│   ├── 열제약매출      = Σ(열제약kWh × MP시간별)
│   └── 용량요금(CP)    = 평일수×평일CP + 휴일수×휴일CP  (발전량 무관)
└── 열판매
    ├── 기본료          = 월별 고정
    ├── 지역사용요금    = Σ(판매Gcal × 단가별)  (주택/업무/공공, 냉방 40%할인)
    └── 인테코          = Σ(인테코판매량 × 판매단가)

[재료비]
├── 가스비
│   ├── CHP  = NG사용량(N㎥) × CHP단가(원/N㎥)
│   ├── PLB  = NG사용량(N㎥) × PLB단가(원/N㎥)
│   └── 기동비용 = 기동시간(h) × 가스사용량(Nm³/h) × CHP단가(원/N㎥)
├── 전력비           = 기본료 + 계량요금(시간별TOU) + 기후환경 + 연료비조정 + 전력기금
├── 용수비           = 정액 + 상수도(2단계) + 하수도(6단계)
├── 약품비           = Σ(6종: 사용량kg × 단가원/kg)
└── 수열비           = Σ(10개열원: Gcal × 구매단가)
```

## 2. 전력판매 계산 상세

### SMP → MP 변환 (Market Price)

```
MP = ROUND(SMP × TLF, 2) + 무부하비정산
```

| 항목 | 출처 | 설명 |
|------|------|------|
| SMP | `smp_hourly` | 시간별 전력거래가격 (원/kWh) |
| TLF | `cp_monthly` [3~5열] | 전력거래손실계수 (평일/토요일/공휴일) |
| 무부하비정산 | 단가표 자동계산 | `ROUND((상수×열량단가)/22000 × 1/(1+2.6327), 2)` |

### 급전매출 (Dispatch Revenue)

```
시간별: hourlyRev = QPC × MW² + LPC × MW + NLPC  (dispatch > 20kW일 때)
일별:   startupRev = 기동비용 (급전 있는 날)
월별:   급전매출 = Σ hourlyRev + Σ startupRev
```

| 파라미터 | 공식 | 설명 |
|---------|------|------|
| QPC | `열량단가 × 1차증분계수` | 2차항 계수 |
| LPC | `열량단가 × 2차증분계수` | 1차항 계수 |
| NLPC | `열량단가 × 상수` | 상수항 |

**고정 계수**:
- 1차증분 = 0.006531
- 2차증분 = 2.232646
- 상수 = 11.855754

### 열제약매출 (Heat Constraint Revenue)

```
열제약매출 = Σ(열제약발전kWh × MP시간별)   (시간별 합산)
```

### 용량요금 (Capacity Charge = CP)

```
용량요금 = 평일일수 × 평일CP + 휴일일수 × 휴일CP
```
- 휴일 = 토요일 + 일요일 + 공휴일
- CP 데이터: `cp_monthly`의 [1열], [2열]

## 3. 가스비 계산

```
CHP단가(원/N㎥) = ROUND(NG단위열량 × (CHP열량단가 - 석유수입부과금) - 안전관리부담금, 2)
PLB단가(원/N㎥) = ROUND(NG단위열량 × PLB열량단가, 2)

CHP가스비 = CHP_NG사용량(N㎥) × CHP단가
PLB가스비 = PLB_NG사용량(N㎥) × PLB단가
```

### 기동비용 (Startup Cost)

```
기동비용 = 기동시간(h) × 가스사용량(Nm³/h) × CHP NG단가(원/Nm³)
```

| 정지 기간 | 분류 | 기본 기동시간 | 기본 가스사용량 |
|----------|------|-------------|---------------|
| ≤8h (1일) | Hot | 4h | 300 Nm³/h |
| ≤48h (2일) | Warm | 6h | 350 Nm³/h |
| >48h (3일+) | Cold | 8h | 400 Nm³/h |

- DP: `totalFuelCost = chpFuelCost + startupCost + plbFuelCost`
- 월별 NG단가가 반영되어 계절별 기동비용 자동 변동

## 4. 열판매매출 계산

### 구조
```
열판매매출 = 기본료 + 지역사용요금

기본료: 월별 고정 (열요금표에서 조회)
지역사용요금: 판매Gcal × 단가 (용도별)
  - 주택용: 단가 그대로
  - 업무용: 난방 단가 / 냉방 단가(=난방×0.6)
  - 공공용: 난방 단가 / 냉방 단가(=난방×0.6)
  - 수영장: 공공용 × 0.5 (수영장비율로 공공난방에서 분리)
```

### 수영장 분리 로직
```
공공난방(일반) = 공공난방합산 × (1 - 수영장비율)
공공난방(수영장) = 공공난방합산 × 수영장비율
수영장 단가 = 공공용 단가 × 0.5
```
- 수영장비율: 열요금표에 월별 소수(0~1)로 입력

## 5. Phase 1 구현 완료 (전력판매 정확화)

| 항목 | 변경 전 | 변경 후 |
|------|---------|---------|
| 전력수익 | `SMP일평균 × MW × 1000` | `Σ(시간별 MP × MW × 1000)` |
| SMP→MP | 미반영 | `MP = ROUND(SMP × TLF, 2) + 무부하비정산` |
| 용량요금 | 미반영 | `평일수×평일CP + 휴일수×휴일CP` |
| 블록배치 | SMP 스코어 | MP 스코어 (TLF 반영) |
| 운영손익 탭 | 미구현 | 월별 공헌이익 테이블 + 요약 카드 |
