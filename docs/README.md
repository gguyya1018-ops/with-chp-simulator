## 핵심 규칙 요약

### CHP 운전
- **연속운전 우선**: 켰다껐다 반복 최소화
- **최소운전시간**: 설정값 (기본 8h)
- **최소정지시간**: 설정값 (기본 4h)
- **부하 범위**: 60% ~ 100%
- **일간 연속성**: 전날 CHP가 24시 근처까지 돌았으면 오늘 0시부터 이어서

### 축열조
- **상한 절대 방지**: storageCap 초과 시 CHP 감축
- **하한 시 PLB 투입**: storageMin 미달 시 PLB

### PLB
- **최후 수단**: CHP/축열조로 불가능할 때만
- **최소운전시간**: 정수 시간 (기본 2h)
- **배치**: 수요 피크 시간대 기반 최적 배치

### 기동비용
- **기동비용** = 기동시간(h) × 가스사용량(Nm³/h) × 월별 NG단가
- Hot(≤1일 정지) / Warm(≤2일) / Cold(3일+) 분류
- DP: 상태에 정지레벨 포함 → 빈번한 ON/OFF 자동 억제
- 휴리스틱: 연속운전 판단 시 기동비용 vs 유지비용 비교

### 비용/수익 계산
```
MP = ROUND(SMP x TLF, 2) + 무부하비정산
열제약매출(일) = Σ(시간별 MP x 송전MW x 1000) (CHP 가동시간만)
용량요금(월) = 평일수 x 평일CP + 휴일수 x 휴일CP
공헌이익 = (열제약매출 + 용량요금) - (CHP연료비 + 기동비용 + PLB연료비)
```
