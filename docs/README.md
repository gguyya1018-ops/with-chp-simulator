## 현재 상태 (2026-03-29)

### 아키텍처
- PyWebView 데스크톱 앱 (frameless, 다크테마)
- 파이프라인: 데이터 → 설정 → 기동계획 → **계산검증** → 운영손익

### 시뮬레이션 엔진
- **MPC v8p8** (CHP 잠금, 해시 검증)
- **PLB v3** (블록탐색, CHP 위에 얹는 구조)
- 결과: UF 0건, OF 20건, 비용 553,816만원

### CHP 운전 원칙
- 하루 1회 기동, 연속 최소 4h
- SMP 높은 시간대 배치
- 주말/공휴일 축열 활용 (금요일 사전 충전)
- 2단계 부하 (고수요=고부하, 저수요=저부하)

### PLB 운전 원칙
- CHP 보조 역할 (CHP 미가동 시 PLB도 미가동)
- 미리 기동 (부족 예상 시점 전 대응)
- 1대 100% 우선, 하루 1회

### 계산검증 (5탭)
- 자체열원: CHP/PLB 가스비+기동비+전력수급+용수약품비
- 외부수열: 열원별 수량×단가 검산
- 열판매 수입: 용도별 Gcal×단가 검산
- 종합 손익: 수입-지출=공헌이익
- 전력수급: 발전/송전/수전 (시간별 모달 포함)

### 전력수급 로직
```
CHP 가동 시:
  발전 = 발전MW × 1000 × 부하율
  송전 = 송전MW × 1000 × 부하율 (한전 판매)
  사내소비 = 발전 - 송전 (건물 전력 자체 커버)
  수전 = 대기전력만 (CHP 커버 불가분)

CHP 미가동 시:
  수전 = 전력수요 모델링 전체 (DH설비+사무실+대기전력)
```

### 비용 계산
```
열제약매출 = Σ(시간별 MP × 송전kWh)
전력비 = Σ(시간별 수전kWh × 전력요금단가)
CHP단가 = (연료비 + 기동비 - 열제약매출) / 생산열량
공헌이익 = (열판매수입 + 열제약매출 + CP) - (CHP가스비 + 기동비 + PLB가스비 + 외부수열비 + 전력비 + 용수약품비)
```
