위드시뮬 PRD

CHP 보일러 기동계획 시뮬레이터 — 제품 요구사항 정의서

v5.3 MPC 정밀최적화 2026-03-20 위드인천에너지

1. 제품 개요

위드인천에너지의 열병합발전(CHP) 및 보조보일러(PLB) 최적 운전 계획을 수립하는 시뮬레이션 도구입니다. 365일 시뮬레이션을 통해 연간 연료비를 최소화하는 기동계획을 자동으로 생성하고, 전력비/가스비/수열비 등 운영비용과 공헌이익을 분석합니다.

3. 설정 (8단계)

Step 1: CHP 사양

부하별 성능표 — 부하율(20~100%), 발전MW, 송전MW, 열출력(Gcal/h), NG소비(Nm3/h), 종합효율(%)

Step 2: PLB 사양

보유대수(기본 3대), 소내전력비, 부하별 성능표

Step 3: 축열조

총용량(1500Gcal), 초기잔량(750), 최소잔량(100), 최대 충/방열 속도

Step 4: 기동/정지 조건

Hot/Warm/Cold Start: 소요시간 × 가스사용량. CHP 최소운전/정지시간(4h), 최소부하(60%). PLB 최소운전시간(2h)

Step 5: 정비일정

최대 3건: 시작일~종료일. 정비일에는 CHP 강제 정지

Step 6: 전력정산 계수

MP = SMP × TLF + 무부하비정산. 1차/2차 증분계수, 상수로 NLPC 계산

Step 7: 외부수열원 관리

EXT_SOURCES 레지스트리: 수열원 추가/삭제/활성화 토글

Step 8: 전력수요 모델링

월별 평균kW × 24시간 패턴 × 보정계수. 실적에서 자동 추출 기능

5. 기동계획 (MPC 최적화)

365일 최적 CHP/PLB 운전 계획을 자동 생성하는 핵심 기능입니다.

최적화 알고리즘: MPC + DP

MPC (Model Predictive Control): 매일 30일 앞을 내다보고 오늘 결정만 확정. 내일은 실제 축열조 잔량으로 다시 계산. 365일 반복.
매일 반복 (1일~365일): ┌─ 제약조건 확정 ──────────────────────────┐ │ 1순위: 정비일 → 가동 불가 │ │ 2순위: minStopTime → 시작시간 제한 │ │ 3순위: overflow → 사전검증 (DP 내부) │ └──────────────────────────────────────────┘ ↓ Block 1: 역방향 DP (30일 윈도우) → 제약 범위 내 최소비용 경로 탐색 ↓ Block 2: 결정 추출 → 오늘의 최적 CHP 시간/부하 ↓ Block 3: 블록 배치 (Stage 2) → MP 높은 시간대에 CHP 배치 ↓ Block 4: PLB 판단 (Stage 3) → 축열조 부족 시 보조보일러 투입 ↓ Block 5: 시간별 시뮬 (Stage 4) → 24시간 에너지 밸런스 검증 ↓ Block 6: 집계 → 비용/수익 산출, 다음날로 상태 전달

7. 운영손익

최종 손익을 4개 카드 + 월별 차트로 시각화합니다.

연간 손익 구조
매출 = 전력판매(CHP발전×MP + 용량요금) + 열판매매출 재료비 = 가스비 + 전력비(수전) + 수열비 + 용수약품비 공헌이익 = 매출 - 재료비 공헌이익률 = 공헌이익 / 매출 × 100

차트: 월별 공헌이익 추이, 이익률 추이, 매출 구성 도넛, 재료비 구성 도넛

8. 비용 계산 체계

전력판매 = CHP발전(MWh) × MP(원/kWh) + CP용량요금(원) MP = SMP × TLF(월별) + 무부하비정산(원/kWh) 열판매 = Σ(항목별 기본료 + 판매량 × 열요금) 연료비 = CHP연료 + PLB연료 + 기동비용 CHP연료 = 가동시간 × NG소비(Nm3/h) × NG단가(원/Nm3) 기동비용 = 기동시간 × 가스사용량 × NG단가 CHP단가 = (연료비 + 기동비 - 전력판매) / 생산열량(Gcal) 수전비 = Σh max(0, 수요h - CHP발전h) × 시간대별전기요금 수열비 = Σ(외부열공급량 × 판매단가) 용수약품비 = 용수사용 × 수도단가 + 약품사용 × 약품단가

10. 사용자 시나리오

신규 프로젝트 생성 → 최적화 실행

1. 홈 → "새 프로젝트" → "24위드" 입력 → 생성 2. 탭1 (데이터) → Excel 일괄 업로드 3. 탭2 (설정) → Step 1~8 입력 → "값 적용" 4. 탭3 (데이터 산출) → "전체 산출" 클릭 5. 탭4 (기동계획) → "최적화 실행" → 365일 MPC 6. 탭5 (비용산출) → 자동 표시 7. 탭6 (운영손익) → 공헌이익 확인

시나리오 비교

1. 설정 변경 (예: CHP 최소부하 60% → 70%) 2. "값 적용" → 탭4 → "최적화 실행" 재실행 3. 결과 비교 분석
