# 위드시뮬 운전최적화 로드맵

## 핵심 원칙
**모든 판단 기준은 비용(Merit Order)**: 열원별 한계비용을 비교하여 가장 경제적인 순서로 투입

## 열원별 비용 구조
| 열원 | 비용 | 매출 | 순비용(원/Gcal) |
|------|------|------|-----------------|
| CHP | NG가스비 | 전기판매(SMP) + 열판매 | (NG비 - 전기수익) / 생산열 |
| PLB | NG가스비 | 열판매 | NG비 / 생산열 |
| 외부수열 | 구매단가 | - | 구매단가 |
| 연료전지(향후) | 연료비 | 전기수익 + 열판매 | (연료비 - 전기수익) / 생산열 |

## 1단계: 일 단위 비용 기반 최적화 (현재)
- [x] 열원별 한계비용(원/Gcal) 계산
- [x] Merit order 적용: CHP 이익 시 최대가동, PLB보다 비싸면 축소
- [x] 비용 추적: 일별 NG비용, 전기수익, 순비용
- [x] 일별/월별 테이블에 비용 컬럼 표시
- [x] 최소정지시간/최소운전시간 제약
- [x] 축열조 오버플로 방지

## 2단계: 시간 단위 최적화
- [ ] 8,760시간(365일×24h) 시뮬레이션 엔진
- [ ] 시간별 SMP 반영 CHP 기동/정지 스케줄
- [ ] 시간별 수요패턴 반영 (열수요 프로파일)
- [ ] 축열조 시간별 충방전 최적화
- [ ] SMP 가중평균 기반 비용 계산

## 3단계: 열원 확장
- [ ] 열원 플러그인 구조 (배열 기반, 추가/삭제 가능)
- [ ] 연료전지 열원 추가
- [ ] 소각열, ERG, SRF 등 외부 열원 개별 모델링
- [ ] 각 열원별 설정 패널 자동 생성
- [ ] 외부수열 계획량 입력 및 단가 반영

## 4단계: 비용 분석 및 시나리오
- [ ] 열원별 비용 breakdown (파이차트, 누적 바차트)
- [ ] NG단가/SMP 민감도 분석
- [ ] 시나리오 비교 (A안 vs B안)
- [ ] 연간 비용 요약 대시보드
- [ ] 열판매 수익 포함 손익 분석
